Node.js Express URL 단축 서비스를 만들어줘. 다음 요구사항으로:

**기능 요구사항:**
- 긴 URL 입력받아서 6자리 랜덤 코드로 단축 URL 생성
- 단축 URL 접속시 원본 URL로 리다이렉트
- 클릭 횟수 카운팅
- 간단한 웹 인터페이스 (URL 입력 폼)

**기술 스택:**
- Node.js + Express
- 메모리 저장소 (일단 간단하게, 나중에 DB 연결 예정)
- EJS 템플릿 엔진 사용

**파일 구조:**
- package.json
- server.js (메인 서버 파일)
- views/index.ejs (메인 페이지)
- public/style.css (간단한 스타일)

**API 엔드포인트:**
- GET / : 메인 페이지
- POST /shorten : URL 단축 생성
- GET /:code : 리다이렉트

**추가 요청:**
- Vercel 배포용 vercel.json 설정 파일도 포함
- 깔끔하고 심플한 UI
- 에러 처리 포함
- README.md 파일도 작성
